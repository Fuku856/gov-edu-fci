# 生徒会活動資金 完全見える化プロジェクト

生徒会の「今」を可視化。予算や議案を通じて、活動を見つめ直してみませんか。

## 概要

このプロジェクトは、生徒会活動資金の使い道を透明化し、生徒全員がアクセスできる情報を提供するためのウェブサイトです。生徒会費の収入と支出を明確に視覚化し、すべての生徒が予算の使途を理解できるようにすることを目的としています。透明性と民主性を確保することで、より信頼される生徒会運営を目指します。

## 主な特徴

- **👁️ 透明性を確保**: すべての会計情報と議案内容を公開し、予算の執行過程や意思決定の流れを明らかにします。隠すことのない、信頼できる生徒会運営を実現します。
- **📊 可視化**: 生徒会の収入・支出をグラフ化。いつでも誰でも確認できる透明性を実現します。
- **💡 わかりやすい報告**: 複雑な会計データを視覚的に表示。予算の使い道が一目でわかり、全校生徒が理解できます。
- **🔍 詳細な内訳**: 各部活動・委員会・行事ごとの予算配分と執行状況を詳細に公開。公平性を保証します。
- **📱 モバイル対応**: スマートフォンからもアクセス可能。いつでもどこでも生徒会の活動をチェックできます。
- **🔒 セキュリティ**: オープンソースの「みらいまる見え」システムを採用。信頼性の高い技術で情報を管理します。

## プロジェクトの目標

1. **透明性の確保**: すべての収入・支出を明細レベルで公開し、生徒会費がどのように使われているかを全校生徒が確認できるようにします。
2. **公平な予算配分**: 各部活動・委員会への予算配分基準を明確化し、申請理由と承認プロセスを可視化します。
3. **生徒参加の促進**: 予算案への質問・意見提出を容易にし、生徒総会での議論を活性化させます。
4. **効率的な運営**: デジタル化により会計処理を効率化し、生徒会役員の負担を軽減します。
5. **他校への展開**: 成功事例として他の学校にも展開可能なモデルを構築し、生徒会改革の輪を広げます。

## 🔐 認証システム

このサイトは学校関係者および開発者のみがアクセスできるよう、**Firebase Authentication**と**Firestoreセキュリティルール**を使用した認証システムが実装されています。

### 認証方法

- **Google認証**: 学校のGoogleアカウント（`@fcihs-satoyama.ed.jp`ドメイン）でログイン
- **GitHub認証**: 開発者用として、許可されたGitHubアカウントでログイン

### アクセス方法

1. サイトにアクセスすると、ログインページが表示されます
2. 学校のGoogleアカウントまたは許可されたGitHubアカウントでログインしてください
3. 認証が成功すると、サイトのコンテンツを閲覧できます

### 認証強化

認証システムは、**Firestoreセキュリティルール**を使用してサーバーサイドで保護されています。

#### 特徴

- **サーバーサイドでの認証チェック**: Firestoreセキュリティルールで保護
- **開発者ツールから回避不可能**: サーバー側で実行されるため
- **管理者権限管理**: Firestoreの `admins` コレクションで管理
- **コード内にメールアドレスを記述しない**: セキュアな実装

#### セットアップ手順

詳細なセットアップ手順は、[Auth/QUICK_SETUP_GUIDE.md](./Auth/QUICK_SETUP_GUIDE.md) を参照してください。

**クイックスタート**:
1. Firestore Databaseを有効化
2. セキュリティルールを設定
3. 最初の管理者を追加

詳細は [Auth/README.md](./Auth/README.md) を参照してください。

- すべてのページはFirebase Authenticationによる認証が必要です
- 学校のGoogleアカウント（`@fcihs-satoyama.ed.jp`）のみがアクセス可能
