# FCI 生徒会活動資金見える化プロジェクト

これは、FCIの生徒会活動資金の使い道を透明化し、生徒全員がアクセスできる情報を提供するためのウェブサイトです。

## 概要

このプロジェクトは、生徒会費の収入と支出を明確に視覚化し、すべての生徒が予算の使途を理解できるようにすることを目的としています。透明性と民主性を確保することで、より信頼される生徒会運営を目指します。

## 主な特徴

- **📊 データの可視化**: 収入と支出をグラフで表示し、直感的に理解しやすくします。
- **💡 わかりやすいレポート**: 会計データを視覚的に整理し、誰でも簡単に確認できます。
- **🔍 詳細な内訳**: 各部活動や委員会、行事ごとの予算配分と執行状況を公開します。
- **🗳️ 民主的な運営**: 予算案への意見提出や質問機能を設け、生徒の参加を促進します。
- **🔒 高い信頼性**: オープンソースの「みらいまる見え」システムを基盤に構築されています。

## プロジェクトの目標

1.  **透明性の確保**: すべての収支を明細レベルで公開します。
2.  **公平な予算配分**: 予算配分の基準を明確にし、プロセスを可視化します。
3.  **生徒参加の促進**: 予算に関する議論を活性化させ、生徒の意見を反映させます。
4.  **効率的な運営**: デジタル化により会計処理を効率化し、役員の負担を軽減します。

## ダッシュボード

ウェブサイトには、実際の収支データや予算の執行状況を確認できるダッシュボードへのリンクが含まれています。
