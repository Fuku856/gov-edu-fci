# 生徒会活動資金 完全見える化プロジェクト

生徒会の「今」を可視化。予算や議案を通じて、活動を見つめ直してみませんか。

## 概要

このプロジェクトは、生徒会活動資金の使い道を透明化し、生徒全員がアクセスできる情報を提供するためのウェブサイトです。生徒会費の収入と支出を明確に視覚化し、すべての生徒が予算の使途を理解できるようにすることを目的としています。透明性と民主性を確保することで、より信頼される生徒会運営を目指します。

## 主な特徴

- **👁️ 透明性を確保**: すべての会計情報と議案内容を公開し、予算の執行過程や意思決定の流れを明らかにします。隠すことのない、信頼できる生徒会運営を実現します。
- **📊 可視化**: 生徒会の収入・支出をグラフ化。いつでも誰でも確認できる透明性を実現します。
- **💡 わかりやすい報告**: 複雑な会計データを視覚的に表示。予算の使い道が一目でわかり、全校生徒が理解できます。
- **🔍 詳細な内訳**: 各部活動・委員会・行事ごとの予算配分と執行状況を詳細に公開。公平性を保証します。
- **📱 モバイル対応**: スマートフォンからもアクセス可能。いつでもどこでも生徒会の活動をチェックできます。
- **🔒 セキュリティ**: オープンソースの「みらいまる見え」システムを採用。信頼性の高い技術で情報を管理します。

## プロジェクトの目標

1. **透明性の確保**: すべての収入・支出を明細レベルで公開し、生徒会費がどのように使われているかを全校生徒が確認できるようにします。
2. **公平な予算配分**: 各部活動・委員会への予算配分基準を明確化し、申請理由と承認プロセスを可視化します。
3. **生徒参加の促進**: 予算案への質問・意見提出を容易にし、生徒総会での議論を活性化させます。
4. **効率的な運営**: デジタル化により会計処理を効率化し、生徒会役員の負担を軽減します。
5. **他校への展開**: 成功事例として他の学校にも展開可能なモデルを構築し、生徒会改革の輪を広げます。

## 🔐 Basic認証の設定

このサイトは学校関係者のみがアクセスできるよう、Basic認証が設定されています。

### 認証情報の取得

アクセスにはユーザー名とパスワードが必要です。認証情報は生徒会執行部またはプロジェクト管理者から提供されます。

### アクセス方法

1. サイトにアクセスすると、認証ダイアログが表示されます
2. 提供されたユーザー名とパスワードを入力してください
3. 認証が成功すると、サイトを閲覧できます

### 管理者向け: 認証設定方法

Basic認証は、**Cloudflare Pages Functions**を使用して実装されています。

**設定手順:**
1. このリポジトリにはすでに `functions/_middleware.js` が含まれています
2. Cloudflareダッシュボードで環境変数を設定（`AUTH_USERNAME` と `AUTH_PASSWORD`）
3. 変更をGitにコミット＆プッシュしてデプロイ

詳細な手順については、[BASIC_AUTH_SETUP.md](./BASIC_AUTH_SETUP.md)または[QUICK_START.md](./QUICK_START.md)を参照してください。
